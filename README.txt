Firma v1.0

--- ES------------------------------------------------------------------------------------------------------------------

La primera versión de Firma está lista para usar. La aplicación en sí se encuentra en la carpeta Firma, dentro de este 
proyecto. Esta carpeta contiene los siguientes elementos:

- Firma1.0.jar -> Aplicación para ejecutar. Requiere Java 11, y se puede ejecutar desde el cmd/bash mediante la sentencia:
"java -jar Firma1.0.jar"

-README.txt -> Fichero que contiene información acerca de Firma.

-resources -> Carpeta que contiene recursos para Firma, ya sean las distintas imágenes cargadas en la aplicación, así como 
una clave privada y un certificado facilitado para probar la aplicación.
